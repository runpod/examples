# Pruna p-image-edit Demos

AI-powered image editing demos using the Runpod AI SDK Provider and Pruna's p-image-edit model.

---

## Setup

1. Install dependencies:

```bash
npm install
```

2. Set your Runpod API key:

```bash
export RUNPOD_API_KEY=your_api_key
```

---

## Demos

### 05 - Virtual Staging

Transform empty or furnished rooms into themed spaces.

```bash
npm run staging:generate
npm run staging:transform
```

See [05-virtual-staging/README.md](05-virtual-staging/README.md) for details.

### 11 - Weather & Time Setter

Change the lighting and atmosphere of outdoor scenes.

```bash
npm run weather:generate
npm run weather:transform
```

See [11-weather-time-setter/README.md](11-weather-time-setter/README.md) for details.
