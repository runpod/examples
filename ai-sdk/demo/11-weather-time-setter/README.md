# Weather & Time Setter

Change the lighting and atmosphere of outdoor scenes using AI-powered image editing with the Runpod AI SDK Provider and Pruna's p-image-edit model.

---

## Usage

1. Generate an outdoor scene:

```bash
node 11-weather-time-setter/generate-scene.js
```

2. Transform the weather and time:

```bash
node 11-weather-time-setter/transform.js
```
